# atom-emmet-rough-parser package

Package written to expand an emmet expression roughly but quickly for readibility. <br />
## commands
<ul>
  <li>_atom-emmet-rough-parser:uglify_ uglify (delete invisibles characters in selection)</li>
  <li>_atom-emmet-rough-parser:beautify_ roughly expand the emmet expansion</li>
  <li>_atom-emmet-rough-parser:deleteHtmlComments_ delete all html comments in selection, sometimes useful</li>
</ul>
